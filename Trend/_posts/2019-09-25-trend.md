## What is a Callback?

## Why do we need Callbacks?

```
function first(){
  console.log(1);
}
function second(){
  console.log(2);
}
first();
second();

```

```
// 1
// 2
```

```
function first(){
  // Simulate a code delay
  setTimeout( function(){
    console.log(1);
  }, 500 );
}
function second(){
  console.log(2);
}
first();
second();
```

```
first();
second();
// 2
// 1
```
